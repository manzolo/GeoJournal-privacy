🔍 Cosa raccoglie GeoJournal

✅ GeoJournal funziona completamente offline senza registrazione. In modalità ospite non viene raccolta né trasmessa alcuna informazione personale.

Solo se scegli di accedere con un account (Google o email), vengono trattati:

  • Email e nome — forniti da Google o inseriti dall'utente, usati solo per identificare l'account
  • Identificatore utente (UID) — generato da Firebase Auth, mai condiviso con terzi
  • Punti geografici — titolo, descrizione, coordinate GPS, tag, emoji, valutazione, URL foto — salvati su Firebase Firestore solo se loggato
  • Foto — caricate dall'utente dalla galleria o dalla fotocamera; rimangono sul dispositivo salvo upload esplicito

Automaticamente (anche in modalità ospite):

  • Report di crash — Firebase Crashlytics raccoglie dati anonimi di crash (modello dispositivo, versione OS, stack trace). Non contengono dati personali.

📍 Posizione GPS

L'app richiede il permesso di localizzazione solo quando l'utente aggiunge un punto e preme "Rileva GPS". La posizione non viene tracciata in background né trasmessa automaticamente.

☁️ Servizi di terze parti

  • Firebase Auth (Google LLC) — autenticazione opzionale
  • Firebase Firestore (Google LLC) — sincronizzazione dati cloud opzionale
  • Firebase Crashlytics (Google LLC) — segnalazioni di crash anonime
  • OpenStreetMap / OSMDroid — tiles cartografiche, nessun dato personale trasmesso
  • Google Drive — usato solo se l'utente configura esplicitamente il backup su Drive tramite la selezione file di Android

Policy di Google: policies.google.com/privacy

💾 Dove vengono conservati i dati

  • Locale — Database Room sul dispositivo (modalità ospite o loggata)
  • Cloud — Firebase Firestore (solo se loggato), server Google nell'UE/USA
  • Backup — File ZIP su storage locale o Google Drive, solo se l'utente lo configura

I dati su Firestore vengono conservati finché l'utente non elimina i punti o il proprio account.

🔒 I tuoi diritti (GDPR)

Se sei un utente dell'Unione Europea hai diritto a:

  • Accesso — sapere quali dati trattiamo
  • Rettifica — correggere dati errati
  • Cancellazione — eliminare i tuoi dati
  • Portabilità — esportare i tuoi dati tramite la funzione "Esporta backup" nell'app
  • Opposizione — interrompere il trattamento disconnettendoti dall'account

Per esercitare i tuoi diritti contattaci tramite GitHub.

🔍 What GeoJournal collects

✅ GeoJournal works fully offline without sign-up. In guest mode, no personal information is collected or transmitted.

Only if you choose to sign in (Google or email):

  • Email and name — provided by Google or entered by the user, used only to identify the account
  • User identifier (UID) — generated by Firebase Auth, never shared with third parties
  • Geo points — title, description, GPS coordinates, tags, emoji, rating, photo URLs — saved to Firebase Firestore only when signed in
  • Photos — uploaded by the user from the gallery or camera; they stay on the device unless explicitly uploaded

Automatically (even in guest mode):

  • Crash reports — Firebase Crashlytics collects anonymous crash data (device model, OS version, stack trace). No personal data included.

📍 GPS Location

The app requests location permission only when the user adds a point and taps "Detect GPS". Location is never tracked in the background or transmitted automatically.

☁️ Third-party services

  • Firebase Auth (Google LLC) — optional authentication
  • Firebase Firestore (Google LLC) — optional cloud data sync
  • Firebase Crashlytics (Google LLC) — anonymous crash reporting
  • OpenStreetMap / OSMDroid — map tiles, no personal data transmitted
  • Google Drive — used only if the user explicitly configures Drive backup via Android's file picker

Google Privacy Policy: policies.google.com/privacy

💾 Where data is stored

  • Local — Room database on the device (guest or signed-in mode)
  • Cloud — Firebase Firestore (only when signed in), Google servers in EU/USA
  • Backup — ZIP file on local storage or Google Drive, only if configured by the user

Firestore data is retained until the user deletes their points or account.

🔒 Your rights (GDPR)

If you are a European Union user, you have the right to:

  • Access — know what data we process
  • Rectification — correct inaccurate data
  • Erasure — delete your data
  • Portability — export your data via the "Export backup" function in the app
  • Objection — stop processing by signing out of your account

To exercise your rights, contact us via GitHub.
